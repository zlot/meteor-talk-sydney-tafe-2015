What Is Meteor?

Meteor is a complete, open source platform for building web and mobile apps in pure JavaScript.

It is a new development environment to take the pain out of building apps for the web.

Why Meteor?

One language everywhere

The same code, written in the same language, can run on both client and server.

Why Meteor?

Pure Javascript

Only one language for the client and server

Code can run on the client or server

Learn one set of APIs / libraries

Why Meteor?

Great Documentation

Why Meteor?

Live Updates

Data updates live on the screen as it changes

i.e., both your changes and the changes made by other users are instantly reflected in the UI

Why Meteor?

Ultra Responsive

Changes made by a user are instantly visible, without having to wait
for a round trip to the server

Why Meteor?

Hot Code Pushes

When new code is released it is automatically injected into
existing clients. No refreshing, no reconnecting!

Why Meteor?

Sensitive Code Protection

Client UI code runs in the browser

Sensitive code runs in a privileged environment on the server

Best practices are baked right into the framework

Why Meteor?

Modular design

Modularity of packages encourage best-practice coding, and make adding new features ultra-simple

Command-line app is a build tool that builds all parts of your app

Why Meteor?

Database integration

A mongoDB database is automatically hooked up on server-side,
and magically reactive on the front-end

Why Meteor?

Accounts

A complete user account system that you can drop
into your application

With one line of code you can have login/logout, account creation, email validation, and login with Facebook, Twitter, etc

Installation

One simple command installs meteor and dependencies!


						$ curl https://install.meteor.com/ | sh

Scared of the command line? Don't be!

Package Management

Adding packages is easy


$ meteor add bootstrap
$ meteor add accounts-ui
$ meteor add fortawesome:fontawesome
$ meteor add jeremy:velocity-animate

Start A New Project

One command to bootstrap a new project!


$ meteor create name-of-project
$ cd meteor
$ meteor

Meteor in action

Meteor comes with complete sample projects useful
for getting started.


$ meteor create --example todo

Conclusion

Meteor is a web framework that takes the pain out of writting web applications.

An entire set of tools to write a reactive web app, that makes the plumbing as simple as possible.

Visit Meteor for more info.
